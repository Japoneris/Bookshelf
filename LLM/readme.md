# Large Language Model Major Papers


## 2017

- [Attention Is All You Need](https://arxiv.org/abs/1706.03762): Founding paper
- [OUTRAGEOUSLY LARGE NEURAL NETWORKS: THE SPARSELY-GATED MIXTURE-OF-EXPERTS LAYER](https://arxiv.org/pdf/1701.06538) MoE premises

## 2019

- [Generating Long Sequences with Sparse Transformers](https://arxiv.org/pdf/1904.10509): How to avoid O(n²) complexity of self-attention - Use a sliding windows + some few tokens of the past


## 2020

- [Scaling Laws for Neural Language Models](https://arxiv.org/pdf/2001.08361)
- [Retrieval-Augmented Generation for Knowledge-Intensive NLP Tasks](https://arxiv.org/abs/2005.11401)
- [An Image is Worth 16x16 Words: Transformers for Image Recognition at Scale](https://arxiv.org/abs/2010.11929)
- [GShard: Scaling Giant Models with Conditional Computation and Automatic Sharding](https://arxiv.org/abs/2006.16668) Mixture of Experts V1
- [Big Bird: Transformers for Longer Sequences](https://arxiv.org/pdf/2007.14062): How to avoid O(n²) complexity of self-attention - Mix between random attention, window attention and global attention over last tokens.
- [Longformer: The Long Document Transformer](https://arxiv.org/pdf/2004.05150): How to avoid O(n²) complexity of self-attention - Use dilated sliding window, to take into account broad local context.

## 2021

- [LoRA: Low-Rank Adaptation of Large Language Models](https://arxiv.org/abs/2106.09685): Finetuning with matrix decomposition
- [QLoRA:  Efficient Finetuning of Quantized LLMs](https://arxiv.org/abs/2305.14314): Finetuning with quantization
- [RoFormer: Enhanced Transformer with Rotary Position Embedding](https://arxiv.org/pdf/2104.09864)
- [Switch Transformers: Scaling to Trillion Parameter Models with Simple and Efficient Sparsity](https://arxiv.org/abs/2101.03961) Mixture of Experts V2


## 2022

- [FlashAttention: Fast and Memory-Efficient Exact Attention with IO-Awareness](https://arxiv.org/abs/2205.14135)


## 2023

- [Flash Attention 2: Faster Attention with Better Parallelism and Work Partitioning](https://arxiv.org/abs/2307.08691)
- [Textbooks Are All You Need](https://arxiv.org/abs/2306.11644): How phi 1 wast trained: small model with accurate data
- [ReAct: Synergizing Reasoning and Acting in Language Models](https://arxiv.org/pdf/2210.03629): Basic Agent programing pattern


## 2024

- [SWE Agents: Agent-Computer Interfaces Enable Automated Software Engineering](https://arxiv.org/abs/2405.15793)


## 2025 

- [Large Language Diffusion Models](https://arxiv.org/pdf/2502.09992): Stable diffusion for text generation

## 2026

- [Humanity last exam](https://arxiv.org/pdf/2501.14249): Dataset with 2500 VERY HARD questions.
